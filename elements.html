Techniques

Caulking techniques vary from person to person. Some prefer using actual tools to smooth out caulking, some prefer to use masking tape, and others prefer to use their hands. Some of these techniques vary by the job type as well, as interior caulking can have significant differences from exterior caulking. Other considerations include the surface being called, preparation involved, and the size of the gap.

Caulking

Timber Tight Sealants almost exclusively uses DOWSIL caulking by DOW. DOWSIL 795 is an ideal mixture for almost any application, while DOWSIL CWS includes an abundance of flavourful colour choices to match your style.
